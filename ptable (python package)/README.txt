PTable: A Python Module for the Periodic Table of Elements

PTable is a Python program that provides a periodic table of elements with properties. This program also allow user to search for an element. 


Additional Library:
This program requires the `Pillow` library. If you don't have it installed, you can do so by running the following command in the Command Prompt:

py -m pip install Pillow


To run the program, execute:

python PTable.py 
or
py PTable.py
